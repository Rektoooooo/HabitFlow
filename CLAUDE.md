# Project Instructions

This is an iOS app built with SwiftUI and SwiftData.

## Task Handling

For every task, use the `ios-developer` skill to pick the best sub-skill for the job. This ensures specialized expertise is applied for:

- UI/layout work → SwiftUI skills
- Data persistence → SwiftData/CoreData skills
- Navigation → Navigation skills
- Animations → Animation skills
- Testing → Testing skills
- And more...

## App Group

The app uses App Group `group.com.habitflow.app` for widget data sharing.
